# Import Test Cases

You can import your test cases into Qase in 3 easy steps.

1. **Click the "..." menu icon** in the top right-hand corner of your repository, and choose "**Import Data**" from the menu.

[![](https://downloads.intercomcdn.com/i/o/wsaz8vex/1244112706/448c34725aacbd4441ff9dd3e4d0/71033.png?expires=1771427700&signature=1feec6978d018a1015a09b3f3b559a1aaecccfe03e1bdc8a4d6af8a4221c66b4&req=dSIjEsh%2Fn4ZfX%2FMW1HO4zWBN8hOKR2QJ55B0U2eKHcNhsSx3%2FlnbwWraJoOP%0Awo1vfJwetvMm%2FyszPsU%3D%0A)](https://downloads.intercomcdn.com/i/o/wsaz8vex/1244112706/448c34725aacbd4441ff9dd3e4d0/71033.png?expires=1771427700&signature=1feec6978d018a1015a09b3f3b559a1aaecccfe03e1bdc8a4d6af8a4221c66b4&req=dSIjEsh%2Fn4ZfX%2FMW1HO4zWBN8hOKR2QJ55B0U2eKHcNhsSx3%2FlnbwWraJoOP%0Awo1vfJwetvMm%2FyszPsU%3D%0A)

After clicking "Import Data," a pop-up will appear.

[![](https://downloads.intercomcdn.com/i/o/wsaz8vex/1244113352/bcf07e3906d70149043a00a0322e/19817.png?expires=1771427700&signature=0fe9c32b0457cf9b0e93766a25cf325c79589fe23adf0a90258bef3f4a855c1e&req=dSIjEsh%2FnoJaW%2FMW1HO4zVTu61njhF%2B0nM0K%2BacGElLkRfdzucLN9OU9Zvor%0A%2FkoAsUFVgeig1DurAzE%3D%0A)](https://downloads.intercomcdn.com/i/o/wsaz8vex/1244113352/bcf07e3906d70149043a00a0322e/19817.png?expires=1771427700&signature=0fe9c32b0457cf9b0e93766a25cf325c79589fe23adf0a90258bef3f4a855c1e&req=dSIjEsh%2FnoJaW%2FMW1HO4zVTu61njhF%2B0nM0K%2BacGElLkRfdzucLN9OU9Zvor%0A%2FkoAsUFVgeig1DurAzE%3D%0A)

2. **Select the Source type** - the TMS platform you are migrating from.

If you're using MS Excel for your test cases, choose the Qase.io format when uploading your file. Ensure that [the headers](https://drive.google.com/file/d/13qYLwBJ6HQ-L_-WpL6Ml8UqgVlharg_6/view?usp=sharing) in your file follow the specified format.

You can download this example CSV template for reference.

[Example CSV](https://drive.google.com/file/d/1QGcRpFcoQk8tJnObh_u7E-gRVjhsJi9c/view?usp=drive_link)

3. **Select a Suite** to act as the parent for all imported test cases. If no suite is defined, all test cases and suites will go to the project's root.

[![](https://downloads.intercomcdn.com/i/o/wsaz8vex/1244113519/1364f75c4999e035bb19c6dda1c1/35622.png?expires=1771427700&signature=88e60b95420b401c66bf904b4e478b0e2f077af1ebcd2aa4a2a59a6ca2d4b211&req=dSIjEsh%2FnoReUPMW1HO4zYs3fdS0igUpmDqIEeH%2FK%2BMz9dcVEkN8PqmvOpNz%0A5ZoF%2BLh8%2FaObKdBh22M%3D%0A)](https://downloads.intercomcdn.com/i/o/wsaz8vex/1244113519/1364f75c4999e035bb19c6dda1c1/35622.png?expires=1771427700&signature=88e60b95420b401c66bf904b4e478b0e2f077af1ebcd2aa4a2a59a6ca2d4b211&req=dSIjEsh%2FnoReUPMW1HO4zYs3fdS0igUpmDqIEeH%2FK%2BMz9dcVEkN8PqmvOpNz%0A5ZoF%2BLh8%2FaObKdBh22M%3D%0A)
