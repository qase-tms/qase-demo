# QQL Widgets

[Saved Queries](https://help.qase.io/en/articles/6417205-saved-queries) can be used to make QQL widgets. Please note, **only Public queries** can be used for QQL widgets.

# Table:

QQL widget allows you to leverage advanced query searches, and pin frequently used or preferred QQL searches to your *[dashboards](https://help.qase.io/en/articles/5563698-dashboards)* to facilitate quicker and more comprehensive monitoring of testing activities.

# Other data visualisations:

For queries that use aggregation and grouping functions, like the example below, you can choose to display the results in formats other than a table.
