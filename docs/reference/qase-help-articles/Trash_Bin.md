From the Trash Bin, you can restore a previously deleted Test Case (either a singular case, or in bulk):

[![](https://downloads.intercomcdn.com/i/o/wsaz8vex/1304802296/5aef4f20b098fdf1a3b1fb7c1c64/20753.png?expires=1771427700&signature=4cbdac704f2208fbcd7c78f5bb9454afe4a4773932e4843ed10e810d03af7caf&req=dSMnEsF%2Bn4NWX%2FMW1HO4zTIBCjhN9jg%2F3XsZGh3NiIzi0QRQ%2BSzjUgjM3a2Q%0ANrstEJ5nY1QgA4JyIhk%3D%0A)](https://downloads.intercomcdn.com/i/o/wsaz8vex/1304802296/5aef4f20b098fdf1a3b1fb7c1c64/20753.png?expires=1771427700&signature=4cbdac704f2208fbcd7c78f5bb9454afe4a4773932e4843ed10e810d03af7caf&req=dSMnEsF%2Bn4NWX%2FMW1HO4zTIBCjhN9jg%2F3XsZGh3NiIzi0QRQ%2BSzjUgjM3a2Q%0ANrstEJ5nY1QgA4JyIhk%3D%0A)

⚠️ Please note that only individual test cases can be restored from the Trash bin. If you’ve **`accidentally deleted a test suite`** containing multiple cases, the test cases can be recovered, but the suite itself must be recreated.

Recovered test cases without an associated suite will be moved to the root of the project under the label “Test cases without a suite.”
