# Notifications

To stay on top of the important updates occurring in your Qase workspace, make sure to set up the notifications.  
​  
To access your notification settings:

1. Navigate to your profile icon on the top right corner of your screen​  
2. In the provided options, look for "Notifications" which should show the screen below.  
​

These settings are for various activities related to test cases, test runs, reviews, and defects.

|  |  |
| --- | --- |
| Entities | Events |
| Test Cases | **a. New Comments on Your Assigned Test Cases:**  Receive notifications when new comments are added to the test cases you are assigned to.    **b. New mention in a comment:**  Receive notifications when you are mentioned in the comments of any test case. |
| Test Plan | **a. When test cases in a plan are Assigned to You:**  Receive notifications when a test cases inside a plan have been assigned to you. |
| Test Runs | **a. Started Test Runs Assigned to You:**  Receive notifications when a test run you have been assigned is started.    **b. If any test cases in a Test Run are Assigned to You:**  Receive notifications when you are assigned to any test cases inside a test run. |
| Reviews | **a. Test Reviews Assigned to You:**  Receive notifications when you are assigned to test reviews. ​  **b. New Comments on Your Assigned Test Reviews:**  Receive notifications when new comments are added to test reviews you are assigned to.    **c. New mention in a comment:**  Receive notifications for only when you are mentioned in the comments of a Review. |
| Defects | **a. New Comments on Your Assigned Defects:**  Receive notifications when new comments are added to defects you are assigned to.    **b. New mention in a comment:**  Receive notifications when you are mentioned in the comments of any Defect. |

Consider triggering a test notification for each category to verify that your email and in-app notifications are set up correctly. Check your email notifications in your email inbox and your in-app notifications through the notification bell on the upper right for the corresponding notifications to confirm that the settings work as expected.
