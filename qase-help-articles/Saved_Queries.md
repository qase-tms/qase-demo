# Saved Queries

***⚠️Queries are available in [Business](https://help.qase.io/en/articles/5563727-business-plan) and [Enterprise](https://help.qase.io/en/articles/6640055-enterprise-plan) subscriptions***

**Queries, based on** [Qase Query Language](https://help.qase.io/en/articles/5563699-queries-qql-qase-query-language), **help you make analytical requests to get specific data from your Qase projects.**

You can save a frequently used query for quick access in the future.

[![](https://downloads.intercomcdn.com/i/o/910440702/506c23d0d2ef67544daa2bde/query+1.gif?expires=1771427700&signature=af07072685496f86d974bc41349414d90d34d85af6ea8c7f4bc5f2f1b6b79565&req=fSEnEs1%2BmoFdFb4f3HP0gAfTlZo30kqfRBKX%2FUICTQ0Exg4TLSWRFjbkBWqO%0Alx1OST7JJiS%2FhMGKwA%3D%3D%0A)](https://downloads.intercomcdn.com/i/o/910440702/506c23d0d2ef67544daa2bde/query+1.gif?expires=1771427700&signature=af07072685496f86d974bc41349414d90d34d85af6ea8c7f4bc5f2f1b6b79565&req=fSEnEs1%2BmoFdFb4f3HP0gAfTlZo30kqfRBKX%2FUICTQ0Exg4TLSWRFjbkBWqO%0Alx1OST7JJiS%2FhMGKwA%3D%3D%0A)

To save a query after searching, simply click on the Save button on the top right.

[![](https://downloads.intercomcdn.com/i/o/894460268/c2a0b1fd491e2bbad61231ce/image.png?expires=1771427700&signature=29d03e72c36bfaad854eae9ee0a1b6c31119af4183f823dea007624f20f533a2&req=fCkjEs9%2Bn4dXFb4f3HP0gL2X1pCwQmTtmOmxpVzQc5w4J86hneS%2BasKcXbDs%0A5AZlfm%2F%2FU2y7tV6JwA%3D%3D%0A)](https://downloads.intercomcdn.com/i/o/894460268/c2a0b1fd491e2bbad61231ce/image.png?expires=1771427700&signature=29d03e72c36bfaad854eae9ee0a1b6c31119af4183f823dea007624f20f533a2&req=fCkjEs9%2Bn4dXFb4f3HP0gL2X1pCwQmTtmOmxpVzQc5w4J86hneS%2BasKcXbDs%0A5AZlfm%2F%2FU2y7tV6JwA%3D%3D%0A)

If you need to search using the same query in future, you can find this saved query in your [Query list](https://app.qase.io/report/queries-list).

[![](https://downloads.intercomcdn.com/i/o/910450502/81742e03ea4e626dcab72cd3/query+3.gif?expires=1771427700&signature=2c8253ed43afd050b1d2a4f6e686b6a4adb73bd2d833cdc8666a75c6c74c0546&req=fSEnEsx%2BmIFdFb4f3HP0gJjhboH5pfPQODVrRpIOwEDQJsy3OoMysnY1mwPK%0Ah05pvmIUEqhIqEfXCw%3D%3D%0A)](https://downloads.intercomcdn.com/i/o/910450502/81742e03ea4e626dcab72cd3/query+3.gif?expires=1771427700&signature=2c8253ed43afd050b1d2a4f6e686b6a4adb73bd2d833cdc8666a75c6c74c0546&req=fSEnEsx%2BmIFdFb4f3HP0gJjhboH5pfPQODVrRpIOwEDQJsy3OoMysnY1mwPK%0Ah05pvmIUEqhIqEfXCw%3D%3D%0A)

When using a Saved Query, the original request is pasted into the field, and you can make changes. After modifying it, you can save it as a new query using "Save As" or update the existing one with the "Update" option.

​

## Public vs Private queries

After creating a query, you can save it as private query (the default option).

[![](https://downloads.intercomcdn.com/i/o/wsaz8vex/2023316573/96f7536b5e7930578948df2ab5a4/49748.png?expires=1771427700&signature=939b88829177875624d31037b219e7d27ef8c1222376baa59680d2d751d3b78b&req=diAlFcp%2Fm4RYWvMW1HO4zXVYUhc46u7ki8xabIMqiHY9xLnO76xWE5Ref0ru%0ANh8lux9f4pOfU1v2jMk%3D%0A)](https://downloads.intercomcdn.com/i/o/wsaz8vex/2023316573/96f7536b5e7930578948df2ab5a4/49748.png?expires=1771427700&signature=939b88829177875624d31037b219e7d27ef8c1222376baa59680d2d751d3b78b&req=diAlFcp%2Fm4RYWvMW1HO4zXVYUhc46u7ki8xabIMqiHY9xLnO76xWE5Ref0ru%0ANh8lux9f4pOfU1v2jMk%3D%0A)

Alternatively, you can make the query public by selecting the checkbox when saving.

Public queries are accessible to all users in your workspace, including both current and future members.

Users with read-only access can also view public queries.

#### Converting existing queries

You can only convert a private query into a public query but not the other way around.

[![](https://downloads.intercomcdn.com/i/o/wsaz8vex/2025918576/7de35b4d26b579bcf3ab24da8dff/37385.png?expires=1771427700&signature=c5fc560cc6a87e18a7a853c14f168d85afe9e3cf287554125036a56b175e7899&req=diAlE8B%2FlYRYX%2FMW1HO4zf7bjHmbhN6MBpVEUcm%2BeRWJP8i3wvCYdP4Dsfe7%0Akf1oqllxviLnFdfu%2Bwc%3D%0A)](https://downloads.intercomcdn.com/i/o/wsaz8vex/2025918576/7de35b4d26b579bcf3ab24da8dff/37385.png?expires=1771427700&signature=c5fc560cc6a87e18a7a853c14f168d85afe9e3cf287554125036a56b175e7899&req=diAlE8B%2FlYRYX%2FMW1HO4zf7bjHmbhN6MBpVEUcm%2BeRWJP8i3wvCYdP4Dsfe7%0Akf1oqllxviLnFdfu%2Bwc%3D%0A)

## Qase Query Language [QQL] widget

Saved (Public) Queries can be used with the [QQL widget.](https://help.qase.io/en/articles/5563698-dashboards#h_9644c6d14d) Private queries cannot be used for QQL widgets.

QQL widget allows you to leverage advanced query searches, and pin frequently used or preferred QQL searches to your *[dashboards](https://help.qase.io/en/articles/5563698-dashboards)* to facilitate quicker and more comprehensive monitoring of testing activities.

[![](https://downloads.intercomcdn.com/i/o/1042565866/bf9a3fa0d91adae26596edfb/qql-widget.gif?expires=1771427700&signature=2be742edebf3a04e71048d4ebe46165168c55b8957a92e0945449eb9d1db79e5&req=dSAjFMx4mIlZX%2FMW1HO4zdNOydQQkjDYvel97pgNOeL%2FmvXTJbJKqTSO1WgO%0AXdMf%2FXmTmYhaiEHsIXA%3D%0A)](https://downloads.intercomcdn.com/i/o/1042565866/bf9a3fa0d91adae26596edfb/qql-widget.gif?expires=1771427700&signature=2be742edebf3a04e71048d4ebe46165168c55b8957a92e0945449eb9d1db79e5&req=dSAjFMx4mIlZX%2FMW1HO4zdNOydQQkjDYvel97pgNOeL%2FmvXTJbJKqTSO1WgO%0AXdMf%2FXmTmYhaiEHsIXA%3D%0A)
