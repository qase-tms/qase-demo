# Configurations

**Configurations** can be used to specify for any [Test Run](https://help.qase.io/en/articles/5563702-test-runs), which hardware or software configuration this run should be performed on.

Configuration groups are found in [Projects](https://help.qase.io/en/articles/5563706-projects):

[![](https://downloads.intercomcdn.com/i/o/1189632920/d7c42a707bc9b35a630e9a24/image.png?expires=1771427700&signature=f35b81981af59496fc2f63cd3ff1bd27bd5dcfecc4a818b85c1fba027a40956b&req=dSEvH899n4hdWfMW1HO4zVLWM52eVU%2BhPym%2BSomD49k3xRphNg4jlsoGrN19%0A9yGBkqn7HGKnHG%2FE%2F1I%3D%0A)](https://downloads.intercomcdn.com/i/o/1189632920/d7c42a707bc9b35a630e9a24/image.png?expires=1771427700&signature=f35b81981af59496fc2f63cd3ff1bd27bd5dcfecc4a818b85c1fba027a40956b&req=dSEvH899n4hdWfMW1HO4zVLWM52eVU%2BhPym%2BSomD49k3xRphNg4jlsoGrN19%0A9yGBkqn7HGKnHG%2FE%2F1I%3D%0A)

Imagine that your team is working on a web application, and you want to test its behavior in several different browsers.

For that, you can create a "Browsers" configuration group and list all the browsers you plan to be performing testing in:

[![](https://downloads.intercomcdn.com/i/o/1189635603/4cc256ad4ce8cdc54c60fe9a/image.png?expires=1771427700&signature=340b7f238dbb71460f884b8da3395bd1613ea84750a372dd10479492b89384d3&req=dSEvH899mIdfWvMW1HO4zd267FIfoLYlxKku%2BHv7Ls%2F0rgLCrwm4d%2B7%2Fj%2F7l%0AJruveoVJlzyEuxePSOc%3D%0A)](https://downloads.intercomcdn.com/i/o/1189635603/4cc256ad4ce8cdc54c60fe9a/image.png?expires=1771427700&signature=340b7f238dbb71460f884b8da3395bd1613ea84750a372dd10479492b89384d3&req=dSEvH899mIdfWvMW1HO4zd267FIfoLYlxKku%2BHv7Ls%2F0rgLCrwm4d%2B7%2Fj%2F7l%0AJruveoVJlzyEuxePSOc%3D%0A)

Once your configuration group has been created, and you've added all possible individual configurations to test on, you can now specify this while preparing a test run - all available configuration groups will be shown in the "Configurations" section:

[![](https://downloads.intercomcdn.com/i/o/1189639528/30b996c0943a04294f597ba4/image.png?expires=1771427700&signature=010350bb39339e7ad2c9d2b6b85fd895a3ec0e3f15d5337c20634fcbf46bbafe&req=dSEvH899lIRdUfMW1HO4zZn9zxzP%2FA3jd%2B5ucH4162%2BhRCGJhcN7E56faO%2B0%0AatE05hm3ydWp0coumTw%3D%0A)](https://downloads.intercomcdn.com/i/o/1189639528/30b996c0943a04294f597ba4/image.png?expires=1771427700&signature=010350bb39339e7ad2c9d2b6b85fd895a3ec0e3f15d5337c20634fcbf46bbafe&req=dSEvH899lIRdUfMW1HO4zZn9zxzP%2FA3jd%2B5ucH4162%2BhRCGJhcN7E56faO%2B0%0AatE05hm3ydWp0coumTw%3D%0A)

You can create multiple configuration groups and have combinations of, for example, different browsers and different OS:
