# Tags

Tags are a workspace level feature, and they allow you to label your [Test Cases](https://help.qase.io/en/articles/5563704-test-cases), [Defects](https://help.qase.io/en/articles/5563710-defects) and [Test Runs](https://help.qase.io/en/articles/5563702-test-runs).

Once a tag has been created, it can be used across all entities and projects.

# Manage Tags

---

All existing tags can be found in [Workspace >> Tags](https://app.qase.io/workspace/tags), where you can also search, filter and delete them in bulk.

On the right, you can see the number of entities each tag has been applied to.

# **Bulk Edit Tags**

---

You can manage tags for multiple test cases simultaneously. With Bulk Edit, you can:

* **Add new tags** to multiple test cases at once.
* **Remove selected tags** from multiple test cases.

[![](https://downloads.intercomcdn.com/i/o/1158054337/720fb5d19d9d662e121e0da7/gif.gif?expires=1771427700&signature=a1f34a25cb51fd102db1bc66bf448b9074275fc54b3cab327bc243ce8814ffdc&req=dSEiHsl7mYJcXvMW1HO4zcJhyDbiYdHwBo%2F6rhWNdrdBzjtqG2lWnQVzpJTO%0ADfdxZZ658hWgFtLWQGk%3D%0A)](https://downloads.intercomcdn.com/i/o/1158054337/720fb5d19d9d662e121e0da7/gif.gif?expires=1771427700&signature=a1f34a25cb51fd102db1bc66bf448b9074275fc54b3cab327bc243ce8814ffdc&req=dSEiHsl7mYJcXvMW1HO4zcJhyDbiYdHwBo%2F6rhWNdrdBzjtqG2lWnQVzpJTO%0ADfdxZZ658hWgFtLWQGk%3D%0A)

If you need to **replace existing tags** with new ones, you can do so by first removing the old tags and then adding the new ones. To make this process easier, your selection of test cases is preserved even after the update.
