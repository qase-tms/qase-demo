# Milestones

A **Milestone** in Qase allows you to create a specific point in your release or development cycle that can be later used to attribute your [Test cases](https://help.qase.io/en/articles/5563704-test-cases), [Defects](https://help.qase.io/en/articles/5563710-defects) and [Test Runs](https://help.qase.io/en/articles/5563702-test-runs). In other words, you can specify which release or version a Test case, Defect or a Test Run "belongs" to, which sprint a Test Run has been executed during, or which sprint a Defect was discovered in.

To create a new Milestone, navigate to "Milestones" section, and hit "Create Milestone":
